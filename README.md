# Getting Started

To started with this Go Project

1. Install Go
2. Make a new directory for this tutorial inside your GOPATH and cd to it. Note: We will use $ to represent the command prompt.

        $ mkdir gowiki
        $ cd gowiki

3. To compile an run the app:  `go build wiki.go` and `./wiki`
4. Start server and go to `http://localhost:8080`
